## 🔧 Основной проект (private)

**Инженерный портал для трассировки трафика и управления инцидентами**  
`Django · DRF · Celery · Redis · PostgreSQL · Linux`

Портал для инженеров крупного провайдера. Решает задачи, которые раньше занимали часы — теперь минуты.

**Что внутри:**
- **Трассировка трафика** — опрос 13 000+ сетевых устройств по SSH, построение маршрута прохождения трафика в реальном времени, быстрое выявление проблемных узлов
- **Обогащение данных** — интеграция с системами мониторинга, конфигурации и справочными БД; актуальные данные всегда под рукой
- **Тикет-менеджмент** — сбор тикетов из TTMS, автоматическая сортировка по приоритетам и типу проблемы; инженеры видят важное первым
- **Форум для согласований** — подсистема обсуждений между группами инженеров при спорных инцидентах
- **Кейсы к вендору** — база обращений с комментариями и результатами; ответы вендора переиспользуются как справочник
- **Утилиты** — генерация конфигураций для тестов, массовый пинг устройств, мелкие автоматизации

**Инфраструктура:** выделенный сервер, администрирую самостоятельно (Nginx, Docker, systemd, бэкапы)
